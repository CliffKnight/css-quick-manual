# <b> Sticky Sidebar 左侧栏固定, 滚动条独立</b>

> 左侧 sidebar 固定在左侧且与视窗同高，当内容超出视窗高度时，在 sidebar 内部出现滚动条。左右两侧滚动条互相独立。

```css
body { 
  height: 100vh; 
  display: flex; 
} 
 
aside { 
  flex: none; 
  width: 200px; 
  overflow-y: auto; 
  display: block; 
  background-color: beige
} 
 
.content { 
  flex: auto; 
  display: flex; 
  flex-direction: column; 
  overflow-y: auto; 
} 
.content article { 
  flex: auto; 
  background-color: red;
}
```
