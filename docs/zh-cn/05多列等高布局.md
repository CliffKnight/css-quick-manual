# <b>多列等宽等高布局</b>



`HTML`

```html
<div class="demo">
    <!--列数：有几个模块-->
    <div class="one">第一个</div>
    <div class="two">第二个</div>
    <div class="three">第三个</div>
</div>
```

`CSS`

```css
.demo {
    height: 200px;
    columns: 3;    /*设置多列的个数,与div个数相同*/ 
}
.one {
    background: beige;
    height: 200px;  /*每个模块的高度必须相同*/
}
.two {
    background: blanchedalmond;
    height: 200px;
}
.three {
    background: beige;
    height: 200px;
}
```
