# <b>黑白图片</b>

![效果](./../assets/color.png)

##### <b>方案：利用`filter`实现。</b>

`HTML`

```html
<img src="img/img-border-demo.jpg">
<img src="img/img-border-demo.jpg" class="gray">
```

`CSS`
```css
img {
    width: 300px;
    height: 300px;
}

.gray {
    -webkit-filter: grayscale(100%);
    -moz-filter: grayscale(100%);
    -ms-filter: grayscale(100%);
    -o-filter: grayscale(100%);
    filter: grayscale(100%);
    filter: gray;
}
```
