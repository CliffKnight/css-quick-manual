# <b>水平右对齐</b>

`HTML`

```html
<div class="box">
    <div class="list">
        <div class="item">居右对齐1</div>
        <div class="item">居右对齐2</div>
    </div>

```

```css

    clear: both;
}

.box {
    width: 760px;
    padding: 10px;

    overflow: hidden;
}

    float: right;
}

.box .list .item {
    display: inline-block;
}
```
