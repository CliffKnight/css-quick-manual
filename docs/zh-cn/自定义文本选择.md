# <b>自定义文本选择</b>

![效果](./../assets/textchoose.png)

##### <b>方案：利用伪元素 `:selection` 实现。</b>


`HTML`

```html
<p class="custom-text-selection">Select some of this text.</p>
```

`CSS`
```css
::selection {
  background: aquamarine;
  color: black;
}
.custom-text-selection::selection {
  background: deeppink;
  color: white;
}
```
