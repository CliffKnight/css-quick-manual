# <b>:非选择器</b>

<b>效果：</b>

![效果](./../assets/not.png)

`HTML`

```html
<ul class="css-not-selector-shortcut">
    <li>One</li>
    <li>Two</li>
    <li>Three</li>
    <li>Four</li>
    <li>Five</li>
</ul>
```

`CSS`
```css
.css-not-selector-shortcut {
    display: flex;
}

li {
    list-style-type: none;
    margin: 0;
    padding: 0 0.75rem;
}

li:not(:last-child) {
    border-right: 2px solid #d2d5e4;
}
```
