### 介绍

黑马CSS代码助手 是一套为后台人员准备的快速上手小册，包含了工作中经常会用到的样式和效果，使用过程中，只需复制文章中的代码，即可实现想要的效果，
这里暂时只列出了笔者在工作中遇到的一些经典效果，如果您有好的建议和意见，请大家一起来完善！

### 功能

- 常见页面布局代码；
- 视觉效果代码；
- 常见动画效果代码；
- 常用交互效果代码；

### 使用

根据页面情况，直接复制`HTML`和`CSS`代码到需要的地方。
